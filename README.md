# TTK Services - Frontend

Application web Angular moderne, scalable et performante pour la gestion des services TTK. Ce projet est propulsé par les standards réactifs d'Angular (Signals, Standalone API, Control Flow) et suit une architecture modulaire et propre (Clean Architecture).

## 🚀 Démarrage Rapide

### Prérequis
- Node.js 18+ 
- npm 9+
- Angular CLI

### Installation

```bash
# Cloner le projet
git clone <repository>

# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm start
